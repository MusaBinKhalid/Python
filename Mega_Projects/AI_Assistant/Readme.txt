The name of the AI Virtual assistant is "Pluto". It responds to your commands after you say the wake word "Pluto".
Before Running the code, make sure to type these commands in the terminal (virtual environment recommended):
    pip install speechrecognition
    pip install pyttsx3
    pip install pyautogui
    pip install pyjokes
    pip install openai (You can use Groq AI, as it offers its API to be used for free)
Add your music folder path in Line 10 of main.py
Add your openAI API key in line 21 of main.py

command examples: 
    pluto open google
    pluto open youtube
    pluto play music
    pluto tell a joke
    pluto type Hello
    Apart from these type of commands, all other commands would be handled by OpenAI ChatGPT.
